## 📅 上架流程时间线

### Google Play
1. 创建开发者账号（$25一次性费用）
2. 创建应用并填写信息
3. 上传APK和素材
4. 填写内容分级问卷
5. 填写数据安全表
6. 提交审核
7. 等待审核（通常1-3天）
8. 审核通过后发布

### 国内应用商店
1. 注册开发者账号（部分需要企业认证）
2. 提交应用信息
3. 上传APK和素材
4. 提交审核
5. 等待审核（通常3-7天）
6. 审核通过后发布

## 🚀 快速上架命令

### 构建Release APK
```bash
cd android
./gradlew assembleRelease
```

### APK位置
```
android/app/build/outputs/apk/release/app-release.apk
```

### 检查APK信息
```bash
aapt dump badging android/app/build/outputs/apk/release/app-release.apk
```
