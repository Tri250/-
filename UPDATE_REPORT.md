## 🎨 技术实现细节

### 健康风险评分算法

```typescript
// 综合评分计算公式
综合评分 = Σ(各维度分数 × 权重系数)

// 示例
体重管理得分: 85 × 0.25 = 21.25
疫苗接种得分: 90 × 0.20 = 18.00
运动锻炼得分: 75 × 0.15 = 11.25
营养状况得分: 80 × 0.15 = 12.00
健康症状得分: 95 × 0.20 = 19.00
定期体检得分: 70 × 0.05 = 3.50

综合评分 = 85.00 分
```

### 症状自查决策树结构

```
[开始]
   ↓
[选择症状部位]
   ↓
┌────────────────────────────────────────┐
│ 消化系统 | 呼吸系统 | 皮肤 | 行为      │
│ 泌尿系统 | 其他                             │
└────────────────────────────────────────┘
   ↓
[具体症状]
   ↓
[严重程度]
   ↓
[持续时间]
   ↓
[诊断结果]
   ├─ 可能情况
   ├─ 严重程度
   ├─ 建议措施
   ├─ 相关手册
   └─ 就医建议
```

## 📱 如何使用新功能

### 1. 健康风险评估
1. 打开应用首页
2. 点击"健康风险评估"卡片
3. 查看综合健康评分
4. 了解各维度详细分析
5. 遵循改进建议

### 2. 症状自查
1. 打开应用首页
2. 点击"症状自查"卡片
3. 选择宠物不适部位
4. 回答引导问题
5. 查看诊断结果和建议

### 3. 健康手册
1. 点击"健康手册"入口
2. 选择分类或搜索
3. 筛选猫咪/狗狗/通用
4. 阅读详细内容
5. 收藏重要手册
