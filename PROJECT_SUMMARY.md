## 项目概述

PawSync Pro 是一个 AI 驱动的宠物健康管理和情感翻译应用，提供以下核心功能：

1. 🐾 **多模态情感翻译** - 语音、图片、视频、文字多模态输入
2. 🏥 **智能健康预警** - 异常指标识别、分级预警、报告生成
3. 💬 **AI 健康咨询** - 智能对话、症状分析、专业建议
4. 📊 **健康数据管理** - 健康记录、疫苗提醒、趋势分析
5. 📱 **移动优化** - Android 无边框适配、离线功能

## 技术架构

### 前端技术栈

- **React 18.3.1** - UI 框架
- **TypeScript 5.8.3** - 类型系统
- **Vite 6.3.5** - 构建工具
- **Tailwind CSS 3.4.14** - 样式框架
- **Zustand 5.0.3** - 状态管理
- **Framer Motion 12.40.0** - 动画库
- **Lucide React 0.511.0** - 图标库
- **Capacitor 8.3.4** - 移动端支持

### 核心服务模块

| 服务 | 文件 | 功能 |
|------|------|------|
| 情感分析服务 | `services/emotionService.ts` | AT-001 ~ AT-032 完整实现 |
| 健康预警服务 | `services/healthWarningService.ts` | A01 ~ F05 完整实现 |
| AI 咨询服务 | `services/aiConsultationService.ts` | E01 ~ E05 完整实现 |
| 认证安全服务 | `services/authService.ts`, `services/securityService.ts` | UC-001 ~ S-006 完整实现 |
| 健康记录服务 | `services/healthService.ts` | 健康数据管理 |
| 宠物服务 | `services/petService.ts` | 宠物信息管理 |

### 类型定义

完整的类型定义位于 `types/` 目录：
- `types/emotion.ts` - 情感分析相关类型
- `types/health-warning.ts` - 健康预警相关类型
- `types/health.ts` - 健康数据相关类型
- `types/ai-consultation.ts` - AI 咨询相关类型
- `types/pet.ts` - 宠物信息相关类型
