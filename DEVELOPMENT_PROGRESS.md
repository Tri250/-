## 技术架构

### 前端技术栈
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS + Framer Motion动画库
- **状态管理**: Zustand
- **UI组件**: 自定义毛球拟态2.0设计系统

### 设计系统
- **毛球拟态2.0**: 柔和、现代、人性化的设计语言
- **色彩系统**: 橙色主色调 + 莫兰迪辅助色 + 全梯度暖灰
- **动效系统**: 物理质感动画（弹性系数、缓动函数）
- **组件系统**: 按钮、卡片、输入框、导航栏、弹窗

### AI功能
- **宠物行为识别**: 12种异常行为检测
- **情感分析**: 声音转文字翻译、情绪识别
- **OCR识别**: 病历文档智能提取
- **AI追踪**: 摄像头画面中宠物实时追踪

### 数据可视化
- **健康仪表盘**: 四维度环形图
- **活动图表**: 弹性动画柱状图
- **睡眠报告**: 环形相位图 + 星星闪烁
- **成长曲线**: 品种对比曲线图
