##### 缺陷1：AI-CHAT-004 健康记录未自动关联

**问题描述**：
当前AI对话接口未自动拉取宠物的健康记录（如体重、疫苗、体检等）注入到Prompt中，导致AI无法基于历史数据进行智能分析。

**问题代码**（[ai.ts:48-50](file:///workspace/backend/src/routes/ai.ts#L48-L50)）：
```typescript
const petInfo = formatPetInfo(pet); // 仅提取基础信息
const aiResponse = await petHealthAI.chat(message, petInfo, conversationHistory);
```

**改进方案**：
```typescript
// 应该拉取宠物完整健康数据
const [healthRecords, vaccines, checkups, growthRecords] = await Promise.all([
  prisma.healthRecord.findMany({ where: { petId }, orderBy: { recordDate: 'desc' }, take: 10 }),
  prisma.petVaccine.findMany({ where: { petId }, orderBy: { date: 'desc' } }),
  prisma.petCheckup.findMany({ where: { petId }, orderBy: { date: 'desc' }, take: 5 }),
  prisma.petGrowth.findMany({ where: { petId }, orderBy: { date: 'desc' }, take: 10 }),
]);

const petInfo = formatPetInfo(pet);
const healthContext = buildHealthContext({ healthRecords, vaccines, checkups, growthRecords });
const fullContext = petInfo + '\n\n' + healthContext;

const aiResponse = await petHealthAI.chat(message, fullContext, conversationHistory);
```

**验收标准**：
- ✅ 自动拉取最近10条健康记录
- ✅ 自动拉取疫苗记录
- ✅ 自动拉取体检记录
- ✅ 自动拉取成长记录
- ✅ 注入到AI Prompt中

#### 需改进项

##### 改进项1：AI-REPORT-002 综合报告异常指标识别

**当前问题**：
健康报告的Prompt结构已完整，但异常指标（如体重骤降）的识别能力依赖于AI模型的推理能力，建议在后端增加预计算逻辑。

**改进方案**：
```typescript
// 在生成报告前，预计算异常指标
function analyzeHealthAnomalies(data: {
  healthRecords: any[];
  vaccines: any[];
  checkups: any[];
  growthRecords: any[];
}) {
  const anomalies: string[] = [];

  // 体重异常检测
  if (growthRecords.length >= 3) {
    const sortedRecords = growthRecords.sort((a, b) =>
      new Date(a.date).getTime() - new Date(b.date).getTime()
    );
    const recentWeight = sortedRecords[sortedRecords.length - 1].weight;
    const previousWeight = sortedRecords[sortedRecords.length - 2].weight;
    if (recentWeight && previousWeight) {
      const changeRate = (recentWeight - previousWeight) / previousWeight;
      if (changeRate < -0.1) {
        anomalies.push(`⚠️ 体重下降超过10%：从${previousWeight}kg降至${recentWeight}kg`);
      }
    }
  }

  // 疫苗过期检测
  const now = new Date();
  vaccines.forEach(v => {
    if (v.nextDate && new Date(v.nextDate) < now) {
      anomalies.push(`🚨 疫苗已过期：${v.name}（应于${v.nextDate}接种）`);
    }
  });

  return anomalies;
}
```

**验收标准**：
- ✅ 后端预计算体重异常变化
- ✅ 后端预计算疫苗过期提醒
- ✅ 异常指标高亮显示在报告中
